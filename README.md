# 🛡️ Kineme - Simple Antivirus & Anti-Malware Simulator

Educational Python project para matuto ng basic virus at malware scanning.

**⚠️ Paalala:** Ito ay **simulation lang** para sa pag-aaral. Hindi ito tunay na antivirus o antimalware.

---

## ✨ Mga Features

- Virus signature scanning
- Malware behavior detection  
- Buong directory / folder scanning
- Madaling gamitin sa Command Prompt / Terminal

---

## 📁 Mga Files

| File                | Gamit |
|---------------------|------|
| `antivirus.py`      | Virus scanner (signatures + extensions) |
| `antimalware.py`    | Malware scanner (suspicious behaviors) |
| `index.html`        | Existing web file |
| `protection 1`      | Iba pang files |
