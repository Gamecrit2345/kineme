# 🛡️ Kineme - Antivirus & Anti-Malware Simulator

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

**Educational Project** — Simple Python-based Antivirus at Anti-Malware Scanner para sa pag-aaral.

---

## ⚠️ Important Disclaimer

> **Ito ay simulation / educational tool lang.**  
> Hindi ito tunay na antivirus at **hindi** ito papalitan ang Windows Defender, Avast, o kahit anong commercial AV.  
> Ginawa ito para matuto ng basic malware detection at Python programming.

---

## ✨ Mga Features

- **Signature-based scanning** (SHA256 + basic hash matching)
- **Heuristic Analysis** (suspicious file behavior at patterns)
- **Multi-file at buong folder scanning**
- **Quarantine system** (nililipat ang suspicious files)
- **User-friendly GUI** (may graphical interface)
- **Command Line** support din
- **Encrypted signature database**
